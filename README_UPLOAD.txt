How to publish on GitHub Pages:

1) Unzip this archive.
2) In your GitHub repo, click: Add file -> Upload files.
3) Drag the CONTENTS of the unzipped folder (index.html, styles.css, assets/, etc.).
4) Commit. Then go to Settings -> Pages -> Source: Deploy from a branch, Branch: main, Folder: /(root).
5) Wait a minute and open your site.
